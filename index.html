Học về Git và GitHub

Bước 1:

Để sử dụng được Git ta cần download: gõ Git Download - > Git -> chọn bản window -> chọn 64 bit để download. Khi download xong mở thư mục bằng Run as admin, các bước tiếp theo chỉ cần chọn Next để cài đặt, sau khi cài xong mở this PC mở ổ đĩa bất kì, nhấn chuột phải nếu có Git Bat và Git GUI là đã cài thành công

Bước 2:

Nếu bạn đã có tài khoản thì chọn sign in, nếu chưa thì chọn sign up để đăng kí.
Để tạo kho chứa dự án: click dấu + phía trên bên phải chọn new repository điền tên dự án, description là tùy chọn, chọn public hoặc private, tích vào Add a README file và chọn Create repository

Bước 3: Upload file lên GitHup

Click vào Add file chọn upload files và kéo thả các file cần upload hoặc copy và paste vào. Trong Commit change ghi nội dung các lần sửa đổi và click Commit change

Bước 4: Deploy(triển khai) website lên online

Vào Setting, chọn tab Pages bên trái chọn branch chọn nhánh main và click save, sau đó chọn tab Action, chọn tiếp vào dự án chọn đường link để mở dự án trên trình duyệt
